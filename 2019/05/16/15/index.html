MAC系统下，Mysql的安装启动及连接

发表于 2019-05-16 | 分类于 program | | 阅读次数:
字数统计: 315 | 阅读时长 ≈ 1

Mysql的安装可以通过两种方法：

  • 命令行(极为不推荐，会遇见许多坑)
  • 官网安装包下载(推荐，成功率高)

下面就主要以官网安装包方法进行安装：

进入官网 ，选择DOWNLOADS——MySQL Community Server，点击下图的Download按钮进行安装包(.dmg)的下载。

下载之后，点击进行安装，不断点击继续按钮即可。此版本8.0.16的mysql已经可以自己设置root账户的密码了。不像旧版本的mysql系统自己生成密码，需要自己记住并保存密码，然而生成的密码常常是不容易记忆的。最新版本的Mysql已经做了更加人性化的改变。

之后就直接出现了这样的界面：

Mysql直接处于启动状态，如果你想关闭可以点击——Stop MySQL Server。

到此，MySQL的安装和启动就已经完成。

如果想再次启动，先找到系统偏好设置。

在最下方能够找到Mysql：

点击即可进入——选择Start MySQL Server，即可启动。

启动完MySQL之后，下面就可以连接MySQL了。

进入终端输入：$ mysql -u root -p

提示输入密码，输入自己设置的密码即可：
